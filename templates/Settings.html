Danger Zone
Irreversible actions — proceed with caution.
Clear all blocked IPs
Removes every manually and auto-blocked IP from the block list. All previously blocked IPs will be allowed again immediately.
Reset all alert rules
Deletes all custom alert rules and resets thresholds to factory defaults. This cannot be undone.
Purge all logs
Permanently deletes all attack logs, IP monitor history, and audit trails. This action is irreversible.
Delete account
Permanently deletes your SecureWatch account, all data, API keys, and configurations. There is no recovery.
